# 家庭打卡提醒系统 - 云端版

一个基于Firebase的家庭任务管理和打卡系统，支持多设备实时同步。

## 功能特性

- 🔐 用户认证（管理员/成员角色）
- 📝 任务管理（创建、编辑、删除）
- ⏰ 智能打卡（时间范围检查）
- 📅 日历视图（任务和打卡记录）
- ☁️ 云端同步（Firebase实时数据库）
- 📱 移动端优化（响应式设计）

## 技术栈

- 前端：HTML5 + CSS3 + JavaScript
- 后端：Firebase (Authentication + Realtime Database + Storage)
- 部署：Vercel + CDN
- 数据库：Firebase Realtime Database

## 快速开始

1. 打开 `family-checkin-cloud.html`
2. 注册管理员账户
3. 创建任务
4. 注册成员账户
5. 查看任务并打卡

## 部署说明

本项目已配置为可直接部署到Vercel，支持中国大陆访问。
